# Lachlan Wellington - Portfolio

Personal portfolio website of Lachlan Wellington, cyber security student at Macquarie University.

🔗 **Live site:** [https://lachlan-wellington.github.io](https://lachlan-wellington.github.io)

## About This Site

A space for cyber security projects, CTF writeups, and learning notes. Built with Jekyll and deployed automatically to GitHub Pages.

## Editing

See `CLAUDE.md` for a full guide to the repo structure, how to add posts and projects, and how deployment works.

To preview locally (requires Ruby and Bundler):

```bash
bundle install
bash tools/run.sh
```

Or just push to the repo - GitHub Actions builds and publishes the site automatically.
